放入 Google Maps API 金鑰

正式部署時建議把金鑰設在 Netlify（見隨附說明），使用者就不必手動輸入。這裡是本機測試用的備用入口；金鑰只存在這台裝置的瀏覽器。

  1. 啟用 Maps JavaScript API 與 Places API (New)
  2. 建立金鑰，用「HTTP 參照網址」鎖定你的 Netlify 網域
找哪種店
挑一種就開動
設好中心，挑一種想吃的 選了分類就自動開鍋搜尋。只端出評論數與評分同時達標的店——人多、分高，被刷單灌水的機率自然低很多。
提示：選分類後依該類人氣排序撈店再合併（高評論數的熱門店優先）。新版 Places API 每次查詢上限 20 筆，市區密集區可縮小半徑、或挑更精確的分類，涵蓋更完整。
